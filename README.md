# Heart Particle Animation 💖

Une animation interactive et fluide d'un cœur en pulsation, générée dynamiquement avec l'API Canvas de JavaScript.

## 🚀 Fonctionnalités
* **Effet de particules :** Des centaines de points lumineux s'assemblent pour former le cœur.
* **Pulsation réaliste :** Le cœur bat en continu avec un effet de halo (glow) dynamique.
* **Interactivité :** Un clic sur l'écran réinitialise l'animation avec une nouvelle dispersion des particules.
* **Design immersif :** Thème sombre (Cyberpunk/Dark) avec intégration d'un overlay affichant un extrait du code source.

## 🛠️ Technologies
* HTML5 / CSS3
* JavaScript (API Canvas / Requêtes d'animation natives)
